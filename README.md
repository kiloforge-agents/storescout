# StoreScout

The in-store layer for brand apps. GPS-guided aisles, AR product overlays,
and geofenced deals — packaged as a progressive web app you ship in days.

This is the marketing site + interactive demo, built with Next.js (App
Router) and Tailwind CSS v4.

## What's inside

- `/` — Marketing home with hero, store map, AR shelf, deals, brands, manifesto
- `/scout` — The interactive in-store mode: live floor map, search, saved list, GPS toggle
- `/ar` — The AR shelf simulator with parallax, scanline, and product reticles
- `/deals` — Geofenced offers, member stacks, wallet passes
- `/brands` — Pricing, pilot application, FAQ
- `/manifesto` — Why we exist

## Run locally

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
```

## Architecture

- All product, aisle, and route data lives in `lib/store-data.ts`
- The `StoreMap` component renders an SVG floorplan with animated routing
- The `ArViewer` component simulates a WebXR AR shelf with pointer/device-orientation parallax
- PWA manifest is generated by `app/manifest.ts`
- Geolocation falls back to a simulated indoor route when permission is denied

## Tech

- Next.js (App Router) + Turbopack
- Tailwind CSS v4 with custom design tokens (Fraunces + Inter + JetBrains Mono)
- Pure client components for interactive demos; everything else is RSC
